# Naming: md-tasks → Schorl

## The Problem

"[Tasks.md](https://github.com/BaldissaraMatheus/Tasks.md)" was already taken, so we needed a new name for this Obsidian Tasks-inspired markdown task manager.

## Requirements

- Two syllables preferred
- Abstract/artsy vibe
- Soft sounds (L, M, W)

## Names Considered

### Nonsense Words
- Lumow, Mowli, Melwa, Wilmo, Welmis — felt too fabricated

### Mineral Theme (to complement Obsidian)
- **Beryl** — gemstone family (emerald, aquamarine, morganite). Clean namespace but common.
- **Bixbite** — red beryl. Nice sound but several tech consulting companies already use it.
- **Perlite** — hydrated obsidian. Perfect thematically, but already an open-source Obsidian markdown viewer at github.com/secure-77/Perlite.

### Black/Silver Minerals (closer to Obsidian's aesthetic)
- Onyx, Jet, Spinel, Galena, Hematite — all considered
- **Schorl** — black tourmaline. One syllable, distinctive, completely clear namespace.

## Final Choice: Schorl

- **What it is**: The mineralogical name for black tourmaline
- **Why it works**:
  - Thematically pairs with Obsidian (both dark minerals)
  - One syllable, easy to type
  - No existing software/apps using the name
  - Distinctive without being unpronounceable
- **Pronunciation**: SHORL (rhymes with "coral")
